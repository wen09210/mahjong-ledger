# 🀄 麻將記帳(Mahjong Ledger)

台灣麻將專用的記帳小工具,打牌時用手機開著,誰胡了按一按,分數自動算好。

**👉 直接使用:https://wen09210.github.io/mahjong-ledger/**

不用安裝、不用註冊,手機瀏覽器打開就能用。紀錄會自動存在你的瀏覽器裡,下次打開還在。

📽️ 第一次用?看這份 [使用簡報](https://wen09210.github.io/mahjong-ledger/slides.html)(方向鍵或滑動翻頁)。

## 使用方式

### 開始之前(設定)

到「**設定**」分頁:

- **玩家名字**:把東、南、西、北改成四位牌友的名字(也可以直接點頂端的分數牌改名)
- **底**:每局的底分,預設 100
- **每台**:一台多少錢,預設 20

### 記帳(三種方式)

打完一局,到「**記一筆**」分頁:

| 方式 | 什麼時候用 | 怎麼算 |
|------|-----------|--------|
| **自摸** | 贏家自己摸到胡牌 | 其他三家各付「底 + 台 × 每台」,贏家收三份 |
| **放槍** | 有人打出的牌被胡 | 放槍的人付「底 + 台 × 每台」給贏家 |
| **手動** | 特殊情況(詐胡、包牌等) | 四家金額自己輸入,每列有 +/− 切換鈕,下方會即時檢查四家合計是否為 0 |

台數用 +/− 按鈕調整;金額預設自動計算,也可以點金額欄直接改(有「恢復自動」按鈕可以還原)。

選好後按「**胡了!記下來**」,頂端四張分數牌立刻更新——**紅色是贏、綠色是輸**(台灣習慣,紅漲綠跌)。

### 歷史紀錄

「**歷史**」分頁可以看每一局的紀錄,記錯了可以單筆刪除,手動輸入的那幾筆會特別標示。

### 共享對局(讓牌友看即時分數)

不用註冊、不用伺服器,透過 WebRTC 點對點直連:

1. 記帳的人:「設定」→「開房間」→ 把 4 位數房號唸給大家
2. 牌友:打開同一個網址 →「設定」→ 輸入房號 →「加入觀看」
3. 房主每記一筆,牌友畫面即時更新(觀看模式為唯讀,不會動到牌友自己裝置上的紀錄)

注意:房主的手機就是資料源,鎖屏或關掉頁面牌友就暫時看不到更新,回來會自動重連。

## 注意事項

- 紀錄存在**各裝置的瀏覽器**(localStorage),手機和電腦的紀錄是分開的,不會同步
- 清除瀏覽器資料會把紀錄一起清掉;換手機前記得先把總帳結清 😄

## 本機開發

純前端、單一元件、零建置流程:React 18 UMD + Babel standalone 在瀏覽器端直接轉譯 [mahjong-ledger.jsx](mahjong-ledger.jsx)。

```bash
# 任何靜態伺服器都行,例如:
python3 -m http.server 8642
# 然後開 http://localhost:8642
```

- [mahjong-ledger.jsx](mahjong-ledger.jsx) — 全部的程式邏輯與畫面(單一檔案)
- [index.html](index.html) — 載入 React、Babel,並用 localStorage 墊片模擬 `window.storage` API
